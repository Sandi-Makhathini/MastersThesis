## About The Analysis

Using citation data from Web of Science (WoS), the aim of this analysis is to investigate how the impact of journal articles from 1962 to 2010. For this investigation, the h and g-indices are calculated. After exporting the data from WoS, the total citations for each article 10 years post publishing is calculated.
